# Compose

Docker Compose stacks used by validator, development, observability, and trainer log shipping workflows.

## Contents

- `docker-compose.yml`: primary validator service stack.
- `docker-compose.dev.yml`: development overrides for local validator work.
- `docker-compose.observability.yml`: validator observability stack.
- `docker-compose.observability-server.yml`: standalone observability server stack.
- `docker-compose.trainer-server.yml`: trainer log shipping stack.
