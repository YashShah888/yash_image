# Trainer Model Prep Tests

Tests for trainer-side model preparation.

## Contents

- `test_comprehensive_stats.py`: comprehensive baseline stats tests; marked `gpu` and skipped unless `--run-gpu` is passed.
- `test_env_stats.py`: environment model-prep stats tests.
- `test_intercode_sidecar.py`: InterCode sidecar/model-prep tests.
- `test_roundtrip.py`: model-prep request/result roundtrip tests.
- `__init__.py`: package marker.
