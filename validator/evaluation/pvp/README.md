# PvP Evaluation

Player-vs-player evaluation runtime for environment tournaments.

## Contents

- `__main__.py`: CLI entrypoint for PvP evaluation.
- `agents.py`: model/player agent wrappers.
- `bot.py`: bot runtime helpers.
- `chat.py`: chat protocol helpers.
- `game_prompts.yml`: game rules and system prompt templates.
- `game_runner.py`: game execution loop.
- `models.py`: PvP evaluation configuration, result, and persistence models.
- `scoring.py`: PvP scoring calculations.
- `server.py`: local PvP server runtime.
- `__init__.py`: package marker.
